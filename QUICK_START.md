# 示範圖集功能 - 快速開始指南

## 🎯 這是什麼？

這是一個完整的「示範圖集管理」功能，讓管理員可以：
1. 📂 從已生成的貼圖中選擇優質作品
2. 📝 自動記錄生成參數（風格、角色、場景、表情）
3. ✨ 在 LINE Bot 和網頁中展示給用戶參考

## ⚡ 3 分鐘快速部署

### 步驟 1: 執行數據庫遷移（1 分鐘）

1. 登入 [Supabase Dashboard](https://supabase.com/dashboard)
2. 選擇你的專案
3. 點擊左側 **SQL Editor**
4. 創建新查詢
5. 複製貼上 `supabase/migrations/20240115_demo_gallery.sql` 的內容
6. 點擊 **Run** 執行

### 步驟 2: 部署代碼（1 分鐘）

```bash
# 在專案根目錄執行
git add .
git commit -m "feat: Add demo gallery feature"
git push origin main
```

Netlify 會自動部署（約 2-3 分鐘）

### 步驟 3: 測試功能（1 分鐘）

訪問：`https://your-site.netlify.app/test-demo-gallery.html`

點擊測試按鈕驗證功能是否正常。

## 📱 如何使用

### 管理員：

1. **登入後台**  
   訪問：`https://your-site.netlify.app/admin/`

2. **進入示範圖集管理**  
   點擊「✨ 示範圖集」卡片

3. **選擇優質貼圖**
   - 在「瀏覽已生成貼圖」分頁
   - 點擊貼圖組的「查看詳情」
   - 滑鼠懸停在圖片上
   - 點擊「➕ 加入示範圖集」

4. **儲存變更**
   - 切換到「當前示範圖集」分頁
   - 檢查已添加的圖片
   - 點擊「💾 儲存變更」

### 用戶：

1. **在 LINE Bot 中輸入：**
   ```
   示範圖集
   ```
   或
   ```
   範例
   ```

2. **查看範例**
   - 左右滑動查看不同範例
   - 每張圖顯示完整參數

3. **創作貼圖**
   - 參考範例的參數
   - 點擊「🚀 開始創建貼圖」
   - 創作類似風格的作品

## 🆘 常見問題

### Q: 沒有看到任何貼圖？
A: 確保你已經創建了一些貼圖組且狀態為「completed」

### Q: 加入示範圖集失敗？
A: 檢查數據庫遷移是否執行成功

### Q: LINE Bot 不顯示？
A: 確保數據庫中至少有一張示範圖

### Q: 參數沒有顯示？
A: 確保創建貼圖時填寫了完整參數
