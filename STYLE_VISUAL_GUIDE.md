## 構圖差異對比

### 全身 (fullbody) 🧍
```
┌─────────────┐
│   👤 15%    │ ← 頭部 (小)
│             │
│   👕 40%    │ ← 上身
│             │
│   👖 30%    │ ← 下身
│             │
│   👟 10%    │ ← 腳部
│   ─────     │
│    5%       │ ← 底部邊距
└─────────────┘

特點: 完整身體，腳必須可見
用途: 動作、跳躍、站立
```

### 半身 (halfbody) 👤
```
┌─────────────┐
│    5%       │ ← 頂部邊距
│   👤 25%    │ ← 頭部 (中)
│             │
│   👕 60%    │ ← 上身+手臂
│             │
│   ─腰─      │ ← 切割線
│    10%      │ ← 底部邊距
└─────────────┘

特點: 腰部以上，手勢清楚
用途: 揮手、指向、手勢
```

### 大頭 (portrait) 😊
```
┌─────────────┐
│    5%       │ ← 頂部邊距
│             │
│   👤 60%    │ ← 頭部 (大)
│             │
│             │
│   👔 25%    │ ← 肩膀
│   ─────     │ ← 切割線
│    10%      │ ← 底部邊距
└─────────────┘

特點: 頭部為主，肩膀可見
用途: 表情特寫、臉部細節
```

### 特寫 (closeup) 👁️
```
┌─────────────┐
│  ─額頭─     │ ← 可能裁切
│             │
│   👁️ 85%   │ ← 臉部 (超大)
│             │
│   👄        │
│             │
│  ─下巴─     │ ← 底部可見
└─────────────┘

特點: 臉部填滿，眼睛中心
用途: 情緒特寫、眼神交流
```

## 組合效果矩陣

### 推薦組合 ✅

| 風格 | 最佳構圖 | 原因 |
|------|---------|------|
| 美顏真實 | 大頭/特寫 | 展現細緻膚質 |
| 可愛風 | 全身/半身 | 展現Q版比例 |

| 風格 | 避免構圖 | 原因 |

| 素描風 | 全身 | 細節線條模糊 |
| 特寫 | 搞笑風 | 無法展現肢體動作 |

## 快速識別指南

### 如何判斷風格是否正確？

#### 美顏真實 ✅
- [ ] 皮膚有柔焦效果
- [ ] 看起來像美顏相機
- [ ] 沒有卡通線條
- [ ] 色彩自然溫暖

#### 可愛風 ✅
- [ ] 眼睛超大（佔臉部40%）
- [ ] 所有東西都是圓的
- [ ] 有粗黑外框
- [ ] 粉嫩馬卡龍色

#### 酷炫風 ✅
- [ ] 有霓虹邊緣光
- [ ] 線條銳利有角度
- [ ] 高對比黑色陰影
- [ ] 霓虹色+黑色

### 如何判斷構圖是否正確？

#### 全身 ✅
- [ ] 腳部完整可見
- [ ] 頭部約15%高度（小頭）
- [ ] 身體佔80%高度
- [ ] 可以看到完整動作

#### 半身 ✅
- [ ] 切在腰部
- [ ] 頭部約25%高度（中頭）
- [ ] 手臂和手勢清楚
- [ ] 肚臍可見

#### 大頭 ✅
- [ ] 頭部約60%高度（大頭）
- [ ] 肩膀可見
- [ ] 臉部表情清晰
- [ ] 脖子完整

#### 特寫 ✅
- [ ] 臉部約85%（超大臉）
- [ ] 眼睛在中心
- [ ] 額頭可能被裁切
- [ ] 下巴可見

## 🎯 實戰技巧

### 風格選擇建議

**想要真實感** → 美顏真實  
**想要萌萌的** → 可愛風  
**想要帥氣的** → 酷炫風  
**想要搞笑的** → 搞笑風  
**想要簡潔的** → 簡約風  
**想要動漫感** → 動漫風  
**想要復古感** → 像素風  
**想要藝術感** → 素描風

### 構圖選擇建議

**展現動作** → 全身  
**展現手勢** → 半身  
**展現表情** → 大頭  
**展現眼神** → 特寫
