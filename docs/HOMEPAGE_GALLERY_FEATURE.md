## 🔧 技術實現

### JavaScript 動態載入

**檔案：** `public/index.html`

**關鍵函數：**

```javascript
async function loadDemoGallery() {
  // 從 Netlify Function 載入示範圖集
  const response = await fetch('/.netlify/functions/demo-gallery');
  const data = await response.json();
  
  // 只顯示前 12 張
  const displayItems = data.items.slice(0, 12);
  
  // 動態渲染 HTML
  container.innerHTML = displayItems.map(item => `
    <div class="group relative bg-white rounded-xl ...">
      <img src="${item.url}" ... loading="lazy">
      <div class="hover-overlay">...</div>
      <span class="badge">${item.styleName}</span>
    </div>
  `).join('');
}
```

### 資料來源

**API：** `/.netlify/functions/demo-gallery`
**資料表：** `demo_gallery`（Supabase）

**欄位：**
- `url` - 圖片網址
- `style` - 風格代碼
- `style_name` - 風格名稱
- `character` - 角色描述
- `expression` - 表情描述
- `display_order` - 顯示順序
