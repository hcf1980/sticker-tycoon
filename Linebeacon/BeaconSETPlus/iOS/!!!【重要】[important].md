###iOS SDK 说明

自2018年3月起，每个版本都会提供两种平台的文件：

**文件名包含“ARM”的为真机发布版本，提交到APPStore时必须使用此版本**；

文件名包含“x86&ARM”的为开发版本，模拟器或者真机均可使用。

此次修改是因为APPStore从2018年3月起不再接受含有x86架构二进制代码的APP。



### iOS SDK Description

From March 2018,  we will provide two files for different platforms in every release version:

**The file name which contains "ARM" for real iPhone device, you must using this version for uploading the APP to the APPStore;**

The file name contains "x86&ARM" for developing, you can using this version for real iPhone device and simulator;

We update this due to From March 2018 The APPStore will not receive the APPs contains the "X86 architecture binary code".
