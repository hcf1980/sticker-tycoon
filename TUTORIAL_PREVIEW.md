## 觸發方式

### 1. 自動觸發
```
用戶發送訊息
    ↓
系統檢查 last_tutorial_shown_at
    ↓
如果 NULL 或 > 7天
    ↓
自動推送：
「👋 歡迎回來！為您準備了完整功能說明，幫助您快速上手 ✨」
    +
功能說明第一部分
```

### 2. 手動觸發
```
用戶輸入：
- 功能說明
- 使用說明
- 教學
- 說明
    ↓
立即回覆功能說明第一部分
```

### 3. 查看第二部分
```
點擊「👉 查看第 2 部分」按鈕
    ↓
發送訊息「功能說明2」
    ↓
收到功能說明第二部分
```

## 設計特點

### 視覺層次
1. **顏色區分**
   - 第一部分：紅色主題 (#FF6B6B) - 代表開始、行動
   - 第二部分：綠色主題 (#06C755) - 代表進階、成長

2. **資訊區塊**
   - 橙色背景 (#FFF3E0)：代幣相關資訊
   - 淺綠背景 (#E8F5E9)：賺取代幣方法
   - 淺紅背景 (#FFEBEE)：重要注意事項

3. **圖示使用**
   - 📖 功能說明
   - 🎨 創建流程
   - 💰 代幣系統
   - 📁 管理功能
   - 🎁 推薦獎勵
   - ⚠️ 注意事項
   - 💡 小提示

### 內容組織
1. **第一部分**：快速上手
   - 核心流程（4步驟）
   - 代幣基礎知識
   - 引導到第二部分

2. **第二部分**：深入了解
   - 管理功能
   - 賺取代幣
   - 注意事項
   - 實用技巧

### 互動設計
1. **按鈕層級**
   - Primary：主要行動（開始創建）
   - Secondary：次要功能（我的貼圖、功能說明）
   - Link：輔助功能（分享賺幣）

2. **引導流程**
   - 第一部分 → 第二部分 → 開始創建
   - 隨時可以跳到其他功能
